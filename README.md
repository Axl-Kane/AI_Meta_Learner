# AI-Powered Deep Learning System (AI极速全景探索与深度攻坚学习系统)

> 一个基于认知科学的AI原生学习操作系统，融合横轴纵轴法与苏格拉底式教学，实现“永不迷路、永不遗漏”的个性化学习。

## 🧠 核心理念
本项目将三大学习理论工程化：
- **横轴纵轴探索法**：纵轴（时间线）建立全局视野，横轴（关键截面）深度剖析，杜绝碎片化。
- **三步极速学习法**：骨架构建 → 主动填肉 → 以教代学，形成完整认知闭环。
- **已知-未知矩阵**：AI主动诊断盲区，系统性发现“未知的未知”。

## 🏗 系统架构（多Agent协同）
>  [用户输入学习主题]
>          ↓
>[探索Agent] ←→ [导师Agent] ←→ [诊断Agent]
> 纵轴/横轴 苏格拉底追问 盲区扫描
>          ↓
>[知识图谱 + 学习报告]

- **探索Agent**：基于MiMo等大模型生成动态时间线与截面分析矩阵。
- **导师Agent**：严格遵循费曼教学法，只提问不直接给答案，迫使主动思考。
- **诊断Agent**：跨学科视角挑战，绘制“无知地图”，照亮认知暗区。

## 🔧 技术栈与Token消耗特征
- **多Agent并发**：单次学习会话需多个Agent协同，产生大量API并行调用。
- **长上下文依赖**：需处理完整知识树、历史对话记录，对100万Token上下文能力依赖度高。
- **高频推理**：每个学习周期涉及几十轮苏格拉底式追问，是典型的“深度推理”场景。
- **目标模型**：小米MiMo-V2.5、Claude 3.5 Sonnet等具备强推理与长上下文能力的模型。
